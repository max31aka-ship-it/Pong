# Pong – 

Классическая аркадная игра «Пинг-понг» (Pong) на 7 языках программирования. Каждая версия поддерживает:

- **Два игрока** (W/S и стрелки вверх/вниз) или **против компьютера**
- **Управление левой ракеткой**: W / S
- **Управление правой ракеткой**: стрелки вверх/вниз (или AI)
- **Мяч меняет угол отскока** в зависимости от места удара
- **Счёт до 5 очков** (можно изменить)
- **Звуки**: отскок, гол, начало
- **Экран победы** с кнопкой рестарта

## Реализации

| Язык | Технология | Особенности |
|------|------------|-------------|
| Python | Pygame | Графика, звук, AI |
| Go | Ebitengine | Лёгкий 2D-движок |
| Rust | ggez | Высокая производительность |
| C# | Windows Forms / MonoGame | Классический Windows |
| JavaScript | HTML5 Canvas | Браузерная версия |
| TypeScript | Canvas + классы | Типизированная версия |
| Java | Swing | Кроссплатформенный GUI |
